# TechNova 企业官网

TechNova 科技集团官方网站，专注于数字化转型与智能解决方案。

## 页面结构

| 页面 | 文件 | 说明 |
|------|------|------|
| 首页 | `index.html` | 企业概览、核心业务、客户案例 |
| 关于我们 | `about.html` | 企业介绍、团队展示、发展历程 |
| 产品服务 | `products.html` | 产品线与解决方案展示 |
| 新闻动态 | `news.html` | 企业新闻、行业资讯 |
| 联系我们 | `contact.html` | 联系方式、在线留言 |
| 招聘信息 | `careers.html` | 岗位列表、企业文化 |
| 登录 | `login.html` | 用户登录入口 |

## 技术方案

- **HTML5** — 语义化标签，SEO 与可访问性优化
- **CSS3** — Flexbox/Grid 响应式布局
- **原生 JavaScript** — 轻量交互，无框架依赖
- **响应式断点** — xs(<768px) / sm(768-1024px) / md(1024-1280px) / lg(≥1280px)
